# Book Library

A simple and interactive **Book Library** built with **HTML, CSS, and JavaScript**, where you can add, remove, and update the reading status of books. 

Books are saved in **localStorage**, so your library stays even after reloading the page!

## Features

- **Add new books** (with title, author, and status)
- **Change reading status** by clicking the status button  
  → cycles between: **Not read → In progress → Read**
- **Remove books** from your library
- **Persistent data** — your books are stored using `localStorage`
- **Default book** (“Thus Spoke Zarathustra” by Friedrich Nietzsche) appears when you first open the app
